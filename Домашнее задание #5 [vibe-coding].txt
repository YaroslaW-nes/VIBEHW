Домашнее задание #5 [vibe-coding]

Задача Взять любую вашу модель из домашек по ML и превратить в работающий веб-сервис 

1. Подготовка: Найдите свой .ipynb файл с обученной моделью (регрессия, классификация, временные ряды и тд) или файл весов (.joblib, .pth).

2. Загрузите ноутбук в Codex/Claude Code, включите режим Plan mode и отправьте промпт:
"Проанализируй этот ноутбук. Выдели только логику предсказания в отдельный файл model.py. Создай класс Predictor с методом predict(data), который загружает веса и возвращает результат. Убери весь код обучения и графиков."

3. Попросите ИИ создать веб-интерфейс: «Создай файл app.py на библиотеке Streamlit. Сделай поля ввода для параметров модели, кнопку 'Рассчитать' и выводи результат из Predictor. Добавь обработку ошибок. Создай файл requirements.txt со всеми нужными версиями библиотек.»

4. Используйте ИИ, чтобы добавить фичей в ваш сервис (не ограничивайте фантазию!). Примеры: интерактивные графики, расчет SHAP values для конкретного наблюдения

5. Попросите ИИ создать Dockerfile для запуска этого Streamlit-приложения

6. GitHub: Создайте репозиторий и запушьте туда ваш проект вместе с Dockerfile

7. Попросите ИИ запустить проект локально (сделайте скрин веб-сервиса) или задеплойте через Timeweb Cloud и отправьте ссылкой.
